Hi Colin, thanks again for the grammar, it will be useful for optimizing Ragel.

This might be what you want:

action call_list {fcall list;}
action return {fret;}

item = word | '(' @call_list;

list := item (
             ( ',' item )* |
             ( '|' item )* )?
         ')' @return;
	
main := item;
