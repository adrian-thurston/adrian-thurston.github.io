>   Attached patch adds ragel ocaml code generation.

>   The only input data type is string (elements are bytes).
>   Several simple tests are included in examples/ocaml/.

> ocaml -T1 : 3.85 sec
> ocaml -F0 : 1.72 sec
> ocaml -F1 : 1.52 sec
> ocaml -G0 : 1.70 sec
> go -G2    : 2.18 sec
>
> rounds : 100_000
> go : 8g version release.2011-02-01.1 7463
